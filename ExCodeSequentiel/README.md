### Goal
Use the script.py file to run n times the CodeSequentiel.c file. 

### How to do ?

* Compile the c file

``` gcc -o CodeSequentiel.c CodeSequentiel ```

* Run the script.py file

``` python3 script.py ```

* Then, give the number of times you want to run the CodeSequentiel.c file.
